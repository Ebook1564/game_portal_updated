Construct 3

What is Construct 3?   

 
It's the game engine used to build this game, and then the game got compiled into HTML5 that you can simply upload to your website
 
This is what the game looks like in the Game Engine:
 
 

 

To modify the game, use the c3p file, the code is organized and grouped for easier modification.
 
Visit editor.construct.net, Construct 3 runs right in your browser! There is nothing to install or set up. If you see an error, check the system requirements page, you might need to update your browser or system.

Game Modifications

Replace Images

Open the Construct 3 project ( c3p ), and double click on any image or sprite.
Click the "Load images from files" button at the top left, Choose your image and close the pop up.
Do this for all the images you want to replace.
NOTE: Make sure your images are the same size ( width and height ) as the original images
 

Sounds & Music

Open the Media folder inside the html5 game, you will find all the sound effects and music files here.

Simply replace them with your own files, making sure you keep the name and format the same!

 

 

 

Variables

Some games have variables that you can edit to change the game experience, like max hp of players or enemies

Always take a look at all the global variables in the event sheets, and read them well to know what they do.

 

 

Changing Texts

To change texts, open the Construct 3 file, and click on any Text object, and on the left side inside the Properties panel you will see "Text" field, change this to whatever text you want. 

 

Publishing

HTML5

Hosting

 
If you want to use the HTML5 Game as it is made, or after you have exported the game from Construct 3, you will need a web server to upload your game to, Go to your File Manager, and go to the "public_html" directory where your websites are usually. Your website folder will exist as a folder with the example name "www.example2.com"
Open the folder and place the content of your HTML5 game there, not the game folder, but its content.
 
 
If you don't have a host, there are many services where you can rent a cheap static file host. The specific steps for uploading files to your server depend on your host. If you need help finding out how to do that, contact your host.
 

 

Embedding 

 
If you want to embed a game inside another web page, you can display it using an iframe element. Simply point the iframe to the web address of your game. For example:
 
<iframe src="https://www.mywebsite.com/mygame/" width="600" height="400" scrolling="no" noresize="noresize" />
 
This will create a fixed size iframe (600x400) inside of which the game will appear. By default the game will automatically scale to fill whatever size the iframe is set to. However if it's a fixed size game, make sure the width and height attributes of the iframe tag match the viewport size of the game.

Tips

  1. Don't rename any file from the game files, if you do that, the game will not launch as it will not find all its elements the way it expects
  2. Don't replce the images with different size images, make sure the image sizes are exactly the same
  3. Some game images have more than 1 sprite merged together, this is Construct 3's way of organizing spritesheets, so if you want to replace a single sprite from the group, make sure you respect its edges and dont go beyond them. Otherwise you will see the images look funny in the game
  4. You can replace sound effects with your own, just make sure the sound files have the same name as the originals
